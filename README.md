## 실행
```bash
# node ver. 18.18.1
# yarn ver. 1.22.21

# yarn 설치되어 있지 않다면
$ npm install -g yarn

# 해당 패키지에서 실행
$ yarn install
$ yarn run dev

# build
$ yarn build
$ firebase deploy
```
