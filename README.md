# About
- React + TypeScript + Vite 를 사용하여
- [https://play2048.co/](https://play2048.co/) 를 참고 한
- `2048 게임` 개발

## 실행
```bash
# node ver. 18.18.1
# yarn ver. 1.22.21

# yarn 설치되어 있지 않다면
$ npm install -g yarn

# 해당 패키지에서 실행
$ yarn install
$ yarn run dev
```
